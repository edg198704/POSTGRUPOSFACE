# Facebook Auto Poster Manual

## Quick Start (WSL/Linux)

**1. Install & Setup**
```bash
git clone <repo_url>
cd <repo_folder>
bash setup.sh
```

**2. Configure Keys**
```bash
nano .env
```
*(Replace `REPLACE_ME` with your Facebook Page Access Token. Save with `Ctrl+O`, Exit with `Ctrl+X`)*

**3. Run the Dashboard**
```bash
bash run.sh
```

**4. Use the Bot**
- The dashboard will open in your browser (usually `http://localhost:8501`).
- Upload an image.
- Write a caption.
- Click **Start Posting**.
